## 🚀 How to Submit Your Project
1. Fork this repository
2. Create folder: `projects/YOUR_GITHUB_USERNAME/`
3. Add your project files + README.md
4. Submit a pull request using the template
5. Tag instructor for review

## ⚠️ Requirements
- All projects must include a working README
- No hardcoded credentials/IPs
- **Include a `.gitignore` file specific to your project's APIs/tools**
- **Include a `requirements.txt` with library names only (no versions)**
